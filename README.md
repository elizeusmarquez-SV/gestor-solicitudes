# Gestor de Solicitudes

Plataforma web desarrollada con Next.js para la gestión de solicitudes

de soporte, permisos y requerimientos.

## Alcance

- Frontend base con navegación

- Pantallas de login, registro y gestión de solicitudes

- Arquitectura de tres capas

- Enfoque serverless

## Tecnologías

- Next.js

- React

- TypeScript

## Estado del proyecto

Semana 1 – Diseño y Frontend base

## Notas finales

Este repositorio sirve como **base de desarrollo** y será extendido progresivamente conforme avance el proyecto, incorporando autenticación, servicios backend y persistencia de datos.
